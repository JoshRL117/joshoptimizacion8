Paquete de optimizacion optimizador-joshua-3
===================================

**optimizador-joshua-3** Este paquete esta enfocado a la optimizacion tanto univarible como multivariable desarrollados mediante métodos desarrollados utilizando el lenguaje de programacion python.
Este paquete tiene metodos para cuando las variables son una sola como son los metodos de optimizacion de busqueda dorada, el método de fibonacci o los métodos basados en gradiente como el metodo de newton-rapshon o el método de la secante. 
Por otro lado, los metodos univaribles estan pensandos cuando se tienen que optimizar mas de una variable, siendo estos muy comunes en la vida real. Este paquete inclue los metodos directos de la caminata aleatoria, hookee jeeves y el método de neldear mead. 
Además, el paquete incluye tambien los métodos basados en gradiente como lo son el metodo cauchy, el gradiente conjugado o fletcher reeves y el del método de newton. Ademas, el paquete incluye funciones de prueba para probar la viabilidad de los algoritmos. 


Para instalar el paquete revise la seccion :doc:`Instalación`
